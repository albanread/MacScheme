MacScheme

Help & Reference

MacScheme combines a Scheme editor, REPL, graphics runtime, sprite tools, and audio/music helpers in one app. This help home is the quickest path into the built-in workflow and the bundled language references.

Start Here
Working in MacScheme

Use the editor pane for source files, the REPL for interactive evaluation, and the graphics/audio procedures for immediate feedback while you iterate.

  • Evaluate forms in the REPL to explore APIs incrementally.
  • Open the graphics screen with gfx-screen, then draw, animate, or create sprites.
  • Use gfx-sprite to place a sprite instance; new instances are visible immediately by default.
  • Use sound-* and music-* procedures for quick effects, playback, and rendering.
  • Browse the full language and Chez implementation docs from the cards below.
