# Local Scheme Docs

This folder contains downloaded HTML documentation for local/offline browsing.

## Books

- [The Scheme Programming Language, Fourth Edition](tspl4/index.html)
- [Chez Scheme User's Guide](csug/csug.html)

## Refresh downloads

Run:

```sh
python3 download_scheme_books.py
```

Or refresh a single book:

```sh
python3 download_scheme_books.py tspl4
python3 download_scheme_books.py csug
```
