MacScheme Guide

Editor Commands

MacScheme’s editor mixes normal macOS document behavior with Scheme-aware structural editing. Most of the advanced commands are available from the Source menu as well as their keyboard shortcuts.

Source Menu highlights

  • Option-Left / Option-Right move backward or forward by one S-expression.
  • Option-Up selects the enclosing form around the cursor.
  • Ctrl-Option-Up splices (unwraps) the enclosing list.
  • Ctrl-Option-Right slurps the next form into the current list.
  • Ctrl-Option-Left barfs the last form back out of the current list.
  • Tab re-indents the current line, and Option-Q re-indents the current selection or current line.

What slurp and barf do

Slurp forward

The current list pulls the next sibling expression inside itself.

(a b) c
=>
(a b c)
Barf forward

The current list pushes its last element back out to the right.

(a b c)
=>
(a b) c
Splice / unwrap

Removes one surrounding list level while keeping the contents.

(a (b c) d)
=>
(a b c d)

Evaluation commands

  • Cmd-E evaluates the current selection, or falls back to the current top-level form.
  • Cmd-Enter evaluates the top-level form around the cursor.
  • Cmd-B evaluates the entire editor buffer.
  • Cmd-. interrupts a long-running evaluation from the Scheme menu.
